# 排盘步骤

## 概述

排盘是八字命理的第一步——将出生时间转化为四柱八字。一个完整的八字排盘含以下信息：

1. 四柱（年、月、日、时）各一个天干+地支
2. 地支藏干（每个地支所暗藏的天干）
3. 十神（以日干为中心标注各天干的关系）
4. 大运（每十年一个大运）
5. 流年（每年的运势变化）

## 步骤一：排年柱

以 **立春** 为分界：
- 立春前出生 → 上一年
- 立春后出生 → 本年

年柱的天干地支直接查干支纪年表。例如 2026 年为丙午年。

## 步骤二：排月柱

以 **节气** 为分界（非农历初一）：

| 月份 | 节 | 气 | 地支 |
|------|------|------|------|
| 正月 | 立春 | 惊蛰 | 寅 |
| 二月 | 惊蛰 | 清明 | 卯 |
| 三月 | 清明 | 立夏 | 辰 |
| 四月 | 立夏 | 芒种 | 巳 |
| 五月 | 芒种 | 小暑 | 午 |
| 六月 | 小暑 | 立秋 | 未 |
| 七月 | 立秋 | 白露 | 申 |
| 八月 | 白露 | 寒露 | 酉 |
| 九月 | 寒露 | 立冬 | 戌 |
| 十月 | 立冬 | 大雪 | 亥 |
| 冬月 | 大雪 | 小寒 | 子 |
| 腊月 | 小寒 | 立春 | 丑 |

月干通过 **五虎遁元**（年上起月法）确定：
> 甲己之年丙作首，乙庚之岁戊为头，丙辛必定寻庚起，丁壬壬位顺行流，若问戊癸何方发，甲寅之上好追求。

即年干为甲或己时，正月为丙寅；年干为乙或庚时，正月为戊寅；依此类推。

## 步骤三：排日柱

日柱需要查万年历或通过公式计算。常用基准算法（2000-2099年适用）：
```
日干支基数 = (年尾二位数 + 7) × 5 + 15 + (年尾二位数 + 19) / 4
```
然后加上当年的第几天，模 60 得干支序号。

## 步骤四：排时柱

先确定时支（出生时辰对应的地支），再用 **五鼠遁元**（日上起时法）确定时干。

五鼠遁元口诀：
> 甲己还加甲，乙庚丙作初，丙辛从戊起，丁壬庚子居，戊癸何方发，壬子是真途。

详见 [[raw/tools/bazi-skill-shichen-table.md]] 的完整对照表。

## 步骤五：排大运

### 顺逆排法
- **阳年男、阴年女**：大运顺排（从月柱的下一个节气开始，顺数到下一个换月节气）
- **阴年男、阳年女**：大运逆排（从月柱的上一个节气开始，逆数到上一个换月节气）

阳年指天干为甲丙戊庚壬的年份，阴年指天干为乙丁己辛癸的年份。

### 起运年龄
以出生日到换月节气的天数 ÷ 3 = 起运岁数（每三天为一年，一天为四个月）。

### 大运干支
顺排：月柱的下一个干支，每十年一运。
逆排：月柱的上一个干支，每十年一运。

详见 [[raw/tools/bazi-skill-dayun-rules.md]] 的详细规则。

## 步骤六：标注十神

以日干为参照，为四柱中每个天干及地支藏干标注十神。详见 [[shi-shen]]。

## 完整排盘示例

以 2026年5月25日 中午12:00 出生为例（公历）：

| | 年柱 | 月柱 | 日柱 | 时柱 |
|--|------|------|------|------|
| 天干 | 丙 | 巳 | 日干 | 午 |
| 地支 | 午 | 月支 | 日支 | 时支 |

（完整排盘需具体日期计算）

此知识后续可结合开源排盘工具（如 [[raw/tools/bazi-skill-wuxing-tables.md]] 来源的 bazi-skill）进行自动化计算。
