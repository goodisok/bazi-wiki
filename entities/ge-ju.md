# 格局

## 概述

格局是八字命理中对命局整体特征的分类型判断。格局决定了命主的先天禀赋、人生走向和事业倾向。格局理论是子平法的核心内容之一。

## 格局分类体系

### 正格（八格、普通格局）

正格指日主与月令的关系确立的八种基本格局。月令即月支，代表命主出生月份所司之五行。

以月令地支的藏干中取格神（通常取月令本气），按十神关系命名：

| 格局 | 格神与日主关系 | 性格特征 |
|------|--------------|---------|
| 正官格 | 月令为正官 | 正直、守规矩、有领导力 |
| 七杀格 | 月令为七杀 | 威严、决断、有魄力、压力大 |
| 正财格 | 月令为正财 | 节俭、踏实、重视财富 |
| 偏财格 | 月令为偏财 | 大方、灵活、商业头脑 |
| 正印格 | 月令为正印 | 仁慈、好学、稳重 |
| 偏印格 | 月令为偏印 | 孤僻、深度思考、特殊才能 |
| 食神格 | 月令为食神 | 温和、才华横溢、享受生活 |
| 伤官格 | 月令为伤官 | 聪明、叛逆、艺术天赋 |

### 变格（外格、特殊格局）

当命局出现极端情况（如一行独旺、无从转化等），不能以正格分析时，入变格：

**从格：**
- 从旺格：全局皆旺于某一行（如木旺极）
- 从弱格：日主极弱无根，全局皆克泄日主

**专旺格（一行得气格）：**
- 曲直格（木旺）、炎上格（火旺）、稼穑格（土旺）
- 从革格（金旺）、润下格（水旺）

**化气格：**
- 天干五合化气成格，如甲己合化土格

## 格局的高低判别

格局本身无绝对好坏，关键在于：
1. **格神清纯**：格局之神不杂不乱（如正官格不混七杀）
2. **用神有力**：格局有喜用神辅助
3. **通关调候**：五行流通、寒暖适中
4. **护卫得当**：格局有印或财的保护

## 格局与用神

格局决定用神的选择：
- 正官格：喜印（印护官）、喜财（财生官）
- 七杀格：喜印（杀印相生）、喜食神（食神制杀）
- 财格：喜官（财生官）、喜食伤（食伤生财）
- 印格：喜官杀（官杀生印）、喜比劫（比劫护印）
- 食伤格：喜财（食伤生财）
