# 滴天髓阐微命例

## 概述

《滴天髓阐微》是清代任铁樵（号铁樵）对《滴天髓》的注解本。与一般注本不同，任铁樵将自己的命理实践经验——数百个真实命例——融入注解中，使理论有了实证支撑。这些命例涵盖三教九流、富贵贫贱，是学习八字命理的绝佳实战教材。

任氏注本之所以影响深远，在于他改变了以往"口诀式"的命理传授方式，代之以"理论+案例"的实证方法。每个案例都包含：八字排盘、旺衰判断、用神取法、大运流年的吉凶应期。
